Creative (/warp creative)

(WorldEdit is on in Creative, within your own plot)

SOME INFO ABOUT WORLDEDIT:
As you know, WorldEdit can get very laggy and sometimes even crash the server.
To solve this problem, I've made it gradually add the blocks, instead of loading them in all at once.
So, if you edited something and it didn't show up, don't try to do it again, it's probably loading in gradually.

The creative part to this server is on a plot system. Everyone gets to have 2 80x80 plots.
If you want, you can merge your plots if they are next to each other, by doing /p merge [direction].
To claim the plot you are standing on, use /p claim.
To automatically retrieve a plot for you, use /p auto.
To clear your plot, use /p clear.
You can see all the other commands by doing /p help.
